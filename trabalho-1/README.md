# Trabalho 1

Google meet clone


## Getting started

Creating the virtual environment

```sh
python -m venv .venv
```

Installing dependencies

```sh
.venv/bin/pip install -r requirements.txt
```

Running the server

```sh
.venv/bin/python ./trabalho-1/server.py
```

Running the client

```sh
.venv/bin/python ./trabalho-1/client.py
```
